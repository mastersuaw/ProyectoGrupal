# Proyecto Grupal — Venta de vehículos

Página web sencilla de catálogo de vehículos en venta (**AutoNorte**): una sola página HTML con estilos y lógica de filtrado incluidos, sin necesidad de instalar dependencias.

## Requisitos

- Navegador moderno (Chrome, Firefox, Edge, etc.).
- Conexión a internet la primera vez que cargues la página (fuentes de Google Fonts e imágenes de ejemplo).

## Cómo ver la página

1. Cloná o descargá el repositorio.
2. Abrí `index.html` con doble clic o arrastrándolo al navegador.

En Windows (PowerShell), también podés ejecutar:

```powershell
Start-Process "index.html"
```

desde la carpeta del proyecto.

## Contenido del proyecto

| Archivo     | Descripción                                      |
|------------|---------------------------------------------------|
| `index.html` | Página principal: listado, filtros y contacto. |

## Personalización rápida

- **Vehículos y precios**: editá el array `vehiculos` en el `<script>` al final de `index.html`.
- **Contacto**: cambiá teléfono, correo y textos en el encabezado, pie y enlaces “Consultar”.
